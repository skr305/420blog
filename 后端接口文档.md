## 项目基本说明

- 项目的测试接口

` http://localhost:8090/`

- 若本次请求正常，则返回`code`为0，`message`为“success”；小于0代表错误, 大于0代表非错误性提示。**任何请求都会返回code和message**，下方接口数据中的**传出参数为data中的讯息**，某些接口没有data数据
- 数据库说明：
  - 数据库名称 ：blog
  - 记得改一下application.properties里的密码

- **统一的错误码**

| 错误码 | 信息        |
| ------ | ----------- |
| 0      | success！！ |
| -1     | 服务器错误  |

- 其他说明
  - 把刚开始建项目时候有关spring security的依赖都先删了
  - 先这些8。，。
